# PhoneCat App

### Version
*Текущая версия: [0.3]()*

### Installation

Cоздать клон репозитория:

```sh
$ git clone https://github.com/Igor-Maf/angular_app
```

Установить [node.js](https://nodejs.org/) и npm:
```sh
$ npm i
```

Переходим в папку с проектом:
```sh
$ cd angular_app
```

В папке проекта устанавливаем bower:
```sh
$ bower install
```

Запустить сервер
```sh
$ npm start
```
